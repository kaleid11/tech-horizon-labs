# Replit Agent Prompt: SEO Migration Fix & Blog Rebuild

## Context

techhorizonlabs.com was rebuilt from WordPress to Replit. The old WordPress site had blog posts and pages that Google still has indexed. During the rebuild, the old blog post URLs were not properly handled — they currently 301 redirect (trailing slash strip) to the same URL without a trailing slash, which returns a 404. This is destroying SEO rankings for pages that were ranking on page 1-2 of Google.

The site also needs www → non-www redirect fixing, schema markup added, and several old blog posts recreated as new `/insights/` pages.

### Implementation Notes

- All redirects must be **301 (permanent)**, not 302
- Both trailing-slash and non-trailing-slash versions of each URL must be handled
- The redirect should happen BEFORE any trailing-slash stripping middleware
- If the app uses a catch-all route or trailing-slash middleware, the redirect map should be checked first

### Suggested Implementation Pattern

If using Express.js or similar:

```javascript
// Place this BEFORE any other route handlers or trailing-slash middleware

const redirectMap = {
  '/ai-business-training-sunshine-coast': '/academy',
  '/ai-consultant-sunshine-coast-business-transformation': '/locations/sunshine-coast',
  '/navigating-the-ai-hype-cycle-turning-ai-potential-into-business-reality': '/insights',
  '/why-your-sunshine-coast-business-needs-a-no-nonsense-ai-consultant-in-2025-breaking-down-the-latest-ai-revolution': '/insights',
  '/master-ai-brain-dumping-sunshine-coast-ai-consultants-2025-guide': '/insights',
  '/ai-consultants-reveal-the-truth-about-privacy-in-the-age-of-artificial-intelligence': '/insights',
  '/your-enterprise-ai-tools-are-probably-overkill-heres-what-queensland-businesses-actually-need': '/insights',
  '/unctad-2025-ai-report-australia-guide': '/insights',
  '/for-home': '/academy',
  '/author/huxleythzn-world': '/about',
  '/event/the-ultimate-ai-assistant-build-day': '/academy',
  '/event/ai-kickstarter-workshop-a-practical-introduction-to-choosing-using-ai-tools-effectively': '/academy',
  '/courses': '/academy',
  '/courses/sample-course': '/academy',
  '/product-category/support': '/contact',
  '/product/free-pre-discovery-call': '/contact',
};

app.use((req, res, next) => {
  // Strip trailing slash for lookup
  const path = req.path.endsWith('/') && req.path !== '/' 
    ? req.path.slice(0, -1) 
    : req.path;
  
  if (redirectMap[path]) {
    return res.redirect(301, redirectMap[path]);
  }
  next();
});
```

If using a different framework, adapt accordingly — the key requirement is:
1. Check the redirect map FIRST
2. Return 301, not 302
3. Handle both `/path` and `/path/` variants

## TASK 2: Fix www Redirect

`https://www.techhorizonlabs.com` currently returns HTTP 503. It must 301 redirect to `https://techhorizonlabs.com` for all paths.

If this is handled at DNS/hosting level (Replit custom domain config), ensure the www subdomain is configured as a redirect to the apex domain. If it needs to be handled in code, add:

```javascript
app.use((req, res, next) => {
  if (req.hostname === 'www.techhorizonlabs.com') {
    return res.redirect(301, `https://techhorizonlabs.com${req.originalUrl}`);
  }
  next();
});
```
